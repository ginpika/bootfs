# TFS 现代化管理员界面使用说明

## 概述

TFS 现代化管理员界面是一套基于 Tailwind CSS 构建的全新管理后台，旨在提供现代化、响应式、用户友好的文件管理体验。该界面完全取代了原有的 Bootstrap 界面，提供了更加美观和高效的用户体验。

## 主要特性

### 1. 现代化 UI 设计
- 采用 Tailwind CSS 构建，遵循现代设计原则
- 响应式布局，支持桌面端、平板和移动设备
- 流畅的交互动效和视觉反馈

### 2. 双视图模式
- **网格视图**：直观的文件卡片展示，适合浏览图片和媒体文件
- **列表视图**：详细的文件信息展示，适合管理大量文件

### 3. 高效的文件操作
- 支持多选操作（批量删除、批量下载、批量发布）
- 快捷键支持（Ctrl+A 全选，Delete 删除等）
- 拖拽上传功能

### 4. 媒体预览能力
- 图片文件支持大图预览
- 视频文件支持播放器预览
- 文件类型图标识别

## 界面布局

### 侧边栏导航
- 文件管理：主文件管理页面
- 画廊：公开图片展示
- MeiliSearch：搜索引擎界面
- 视频播放器：视频文件播放
- 相册：相册展示

### 顶部工具栏
- 搜索框：实时搜索文件
- 视图切换：网格/列表视图切换
- 刷新按钮：手动刷新文件列表
- 上传按钮：上传新文件

### 主要功能区
- 统计卡片：显示文件总数、图片数、视频数等
- 操作工具栏：各种文件操作按钮
- 文件列表：展示文件网格或列表

## 功能详解

### 文件操作
1. **上传文件**：点击上传按钮或拖拽文件到页面
2. **选择文件**：点击文件卡片或复选框进行选择
3. **预览文件**：点击图片缩略图或预览按钮
4. **下载文件**：点击下载按钮或右键菜单
5. **删除文件**：点击删除按钮或使用批量删除

### 批量操作
- 选中多个文件后，顶部会出现批量操作工具栏
- 支持批量下载、批量删除、批量发布到图床
- 使用 Ctrl+点击可选择多个不连续的文件

### 文件筛选
- 搜索框支持文件名搜索
- "只显示主存储"按钮过滤副本文件
- 状态标签快速筛选（HLS、相册、画廊等）

## 快捷键

- `Ctrl+A` / `Cmd+A`: 全选当前页面文件
- `Delete`: 删除选中的文件
- `Esc`: 关闭模态框或取消选择

## 技术架构

### 前端技术栈
- **Tailwind CSS**: 原子化 CSS 框架
- **原生 JavaScript**: 无框架依赖
- **HTML5**: 现代化语义化标签
- **响应式设计**: 移动优先的设计理念

### 后端集成
- 完全兼容原有的 TFS API
- 保留所有原始功能（上传、下载、删除、转换等）
- SSO 集成保持不变

## 故障排除

### 常见问题
1. **样式不生效**：检查浏览器缓存，尝试强制刷新
2. **JavaScript 错误**：检查浏览器控制台错误信息
3. **文件上传失败**：检查网络连接和服务器配置
